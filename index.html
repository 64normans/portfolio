work

These are all my past projects that I've worked on over the years. They include both school work as well as my own personal projects. I made many of my personal projects to explore and learn more about a particular technology or tool so as to deepen my understanding of the mechanics.



Bookstore Website


This project centers around an e-commerce website selling ebooks. It was a team effort where I was primarily designing the theme of the website and also some of the functions such as admin accounts and management.


As you can see, we chose to go with a black and orange colour theme as it looked appropriate in our opinion, a nice contrast from the all white background. We made the navigation bar at the top follow the screen so that no matter how low the user scrolls down, they would always have the navigation bar to help them navigate towards somewhere else.





Video Game Guide Website


This earlier work aims to create a website similiar to a blogsite, where the owner writes reviews and guides on a specific video game. Whereas the bookstore focuses on functions such as shopping cart and displaying data, this website focuses on organisation and formatting of website layout and content.


This was the first website I made without CSS templates or bootstrap, simply my raw input in coding the files. Therefore, this is a display of how far I can get without a scaffolding.





Pokemon Game Program


This project differes in the way that it is not a website, and instead is a C# program that demonstrates a simple console video game created using C#. The game behaves like a text-based program that is supposed to mimic a Pokemon device that stores various Pokemons and index them.


Main focus is on object implementation in C# language with persistence, meaning the program is able to create files to store data even after program has ended.
